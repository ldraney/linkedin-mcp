Install

Multiple ways to get started. Authentication can be configured after install.

Claude Desktop (.mcpb)

  1. Download ldraney-linkedin-mcp.mcpb
  2. Double-click the file to install in Claude Desktop
  3. Enter your LinkedIn credentials when prompted, or use the OAuth tools to authenticate

Claude Code / CLI

Add to your MCP config (~/.claude/settings.json or project settings):

{
  "mcpServers": {
    "linkedin": {
      "command": "uvx",
      "args": ["ldraney-linkedin-mcp"],
      "env": {
        "LINKEDIN_ACCESS_TOKEN": "your_token",
        "LINKEDIN_PERSON_ID": "your_person_id"
      }
    }
  }
}

Or install with pip: pip install ldraney-linkedin-mcp

Authentication options: You can set LINKEDIN_ACCESS_TOKEN and LINKEDIN_PERSON_ID as environment variables, store them securely via the save_credentials tool (OS keychain), or use the built-in OAuth flow with get_auth_url.
